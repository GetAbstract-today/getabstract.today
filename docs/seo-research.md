## 3. SEO Title Best Practices

### Optimal Format
```
[Primary Keyword] - [Value Prop/Hook] | [Brand Name]
```

### Rules
1. **Length:** 50â€“60 characters (Google truncates at ~60). Aim for 55.
2. **Front-load primary keyword:** Google gives more weight to words at the start.
3. **Include brand name:** At the end, after a pipe `|` or dash `â€“`.
4. **Power words that drive clicks:**
   - Free, Daily, Best, Top, Quick, Essential, Curated
   - Action verbs: Get, Join, Subscribe, Discover
   - Urgency/FOMO: Today, Now, Don't Miss, Never Miss
   - Numbers: "2-Minute", "5-Minute", "100+ Sources"
5. **Avoid:**
   - Stuffing multiple keywords unnaturally
   - ALL CAPS
   - Repeating brand name
   - Generic titles with no differentiation

### Character Breakdown
```
|---- Primary keywords (25-30 chars) ----|-- Hook/Differentiator --|-- Brand --|
"Free Daily AI & Tech Newsletter          - 2-Min Reads              | Abstract"
```

## 4. Meta Description Best Practices

### Optimal Format
```
[Value proposition with primary keyword]. [Secondary keyword + differentiator]. [Social proof]. [CTA].
```

### Rules
1. **Length:** 150â€“160 characters. Google may truncate beyond 160.
2. **Primary keyword** within first 60 characters (visible in all SERPs).
3. **Secondary keywords** naturally woven in (AI, startups, tech, daily, email).
4. **CTA:** "Subscribe free", "Join X readers", "Sign up today".
5. **Social proof:** Reader count, freshness ("Updated daily"), authority.
6. **Urgency/Value:** "in 2 minutes", "free forever", "never miss a trend".
7. **Avoid:**
   - Keyword stuffing
   - Duplicate of title
   - No period/sentence structure (looks spammy)
   - Exceeding 160 chars significantly
